Affiche Du Festival MMI

Année : 2024 / 2025

Description succincte du projet :
Dans le cadre d’un projet Universitaire nous avons réalisé les affiches promotionnelles du festival MMI A L’West Fest.

Cadre : Réalisé dans le cadre de la formation MMI

Logiciels : Photoshop, Illustrator

Résultat : Fini

Aptitudes développées :

  • Organisation
  • Faire des recherches graphiques
  • Créativité
Premiere Pro / After Effect / Flipaclip / FlipNote 3D / Kinemaster / DaVinci Resolve / Photoshop / Illustrator / AffinityDesigner / AffinityPhoto / Blender / Unity / UnrealEngine / PHP / MySQL / SQL / phpMyAdmin / HTML / CSS / JAVASCRIPT / Bulma / Bootstrap / C / C++ / C# /

Mes recherches

Durant le projet j'ai du faire des recherches d'idée :

  • Au cours de cette étape, j'ai exploré différentes idées pour le projet. J'ai étudié les différentes recherches graphiques des autres IUT, pour déterminer les éléments qui les démarquent. A la suite de cette recherche, plusieurs idées sont venues à l'esprit.
  • Je me suis donc dirigé vers des idées qui mettaient en relation le patrimoine breton et la création multimédia
Mes idées :
  • Ma première idée était un sac géant dans lequel il y avait les matériaux des différentes catégories du festival A l'West Fest
  • Ma deuxième idée était du photomontage avec les célèbres maisons de Lannion pour rappeler le patrimoine de la localisation de l'IUT. Et en arrière-plan, une explosion de couleur avec les éléments qui représentent les catégories du festival

Résultat Final

Mon choix final est un bol breton (en rapport avec le patrimoine breton) qui est rempli de créativité, avec une explosion de couleur, avec les différents éléments des catégories du festival
J'ai choisi un style assez enfantin et en gribouilli pour reporter à la catégorie graphisme du festival avec des couleurs vives et harmonieuses pour faire ressortir les éléments en violet, ce qui ramène à la couleur de la formation MMI qui est violette, et aussi le bol breton qui est bleu, pour faire ressortir le patrimoine breton
